# Tech Stack

本项目使用以下技术栈：

## 核心技术

| 技术                       | 版本 | 说明                         |
| -------------------------- | ---- | ---------------------------- |
| React                      | 18.x | UI 框架                      |
| TypeScript                 | 5.x  | 类型系统                     |
| Umi Max                    | 4.x  | 应用框架（路由、构建、部署） |
| Ant Design                 | 5.x  | UI 组件库                    |
| @ant-design/pro-components | 2.x  | 高级组件库                   |

## 请求与状态

| 技术            | 说明                    |
| --------------- | ----------------------- |
| Umi Max Request | 基于 axios 封装的请求库 |
| React Hooks     | 状态管理                |

## 样式方案

| 技术            | 说明           |
| --------------- | -------------- |
| Less            | CSS 预处理器   |
| CSS Modules     | 样式模块化     |
| Ant Design 样式 | 组件库自带样式 |

## 开发工具

| 工具        | 说明         |
| ----------- | ------------ |
| ESLint      | 代码检查     |
| Prettier    | 代码格式化   |
| Husky       | Git 钩子     |
| Lint Staged | 暂存文件检查 |
